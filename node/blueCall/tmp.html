  • Methods

    • POSTGet Audience Reach
      To check the inventory for an audience, copy one of the provided JSON Segment Reach examples, paste it into the Body box, and then modify the segments and categories.

      JSON Segment Reach Examples

      Single Segment with Single Category
      {"AND": [{"OR": [{"cat": 17}]}]}

      Single Segment with Multiple Categories
      {"AND": [{"OR": [{"cat": 7624},{"cat": 7628}]}]}

      Multiple Segments with Single Categories
      {"AND": [{"OR": [{"cat": 139}]},{"OR": [{"cat": 6089}]}]}

      Multiple Segments with Multiple Categories
      {"AND":[{"OR":[{"cat":2150},{"cat":355}]},{"OR":[{"cat":2095},{"cat":2098}]}]}

      Multiple Segments with Frequency
      {"AND": [{"OR": [{"cat": 139,"freq":[5]}]},{"OR": [{"cat": 6089, "freq":[5, 10]}]}]}

      Excluded Segment
      {"AND": [{"OR": [{"cat": 18}]},{"NOT": {"OR": [{"cat": 71},{"cat": 5814}]}}]}

      ParameterValueTypeDescription
      bodytextarea

      Include the following parameters in the POST request:

      segment: An object representing your audience with the following structure

      - one AND object representing the audience for which you want to get the reach

      - A list of one or more OR objects in the AND object representing the segments in the audience

      - Each segment may optionally include a freq parameter (frequency) specifying the minimum number of times that users must have qualified for a category since they were initially tagged with it to be included in the segment. This may be a list with one integer (for example, [5]) specifying the minimum frequency, or a comma-separated list of two integers (for example [5, 10]) specifying the minimum and maximum frequency.

      - A list of one or more category numbers (cat) in each OR object representing the categories in the segment

      - optionally, one NOT object for any categories to be excluded from the audience. This object has a similar structure as the AND object except you can only have one OR object in it.

      See the Audience Creation Reference for examples of the segment object that you can use to create your audience.

      deviceTypeIDint

      Optional. An integer specifying on which devices you want to target users (the default device depends on your DMP). Enter one of the following values:

      0 All (Desktop and Mobile)
      1 Desktop
      2 Mobile
